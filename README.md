# Hashtable Extensível

Para compilar o programa, é preciso rodar o seguinte comando:

```bash
dotnet build
```

Para apenas rodar o programa, é preciso executar o seguinte comando:

```bash
dotnet run --project src/HashTableExtensivel/HashTableExtensivel.csproj -- [Parâmetros]
```

Para melhor documentação do programa de Console, execute o programa com o parâmetro `--help`:

```bash
dotnet run --project src/HashTableExtensivel/HashTableExtensivel.csproj -- --help
```
